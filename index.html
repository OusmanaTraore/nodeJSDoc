Commencer à utiliser NodeJs

Création d'un serveur web , Feb 21 , 2021

Après installation du framework NodeJs , créons notre premier serveur web.

Etape:1

Créer un fichier avec une extension .js ; exemple "app.js" puis insérer les lignes de codes suivantes

const http = require('http');
const hostname = '127.0.0.1';
const port = 3000;
const server = http.createServer((req, res) => {
res.statusCode = 200;
res.setHeader('Content-Type','text/plain');
res.end('Hello World');
});
server.listen(port,hostname, () => {
console.log(`Server running at http://${hostname}:${port}/`);
});

Etape:2

Lancer le serveur avec:
node app.js

Etape:3

Visiter votre serveur web à l'adresse:
http://localhost:3000

Vous devez voir un message de bienvenu:
Hello World
